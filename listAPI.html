List of APIs for LightBank project

Get all accounts: GET /accounts

Return list of all accounts in system

Response on success:

Status: 200

JSON:

  1. [{
  2. number: string,
  3. owner: string,
  4. sum: number
  5. }, ….]

Response on error:

Status: 404

Get account: GET /account/{accNumber}

Return all data for account number {accNumber}

Response on success:

Status: 200

JSON:

  1. {
  2. number: string,
  3. owner: string,
  4. sum: number
  5. }

Response on error:

Status: 400

String: Error message

Add new account: POST /account

Create new account with owner’s data from request’s body

Request body:

  1. {
  2. name: string
  3. }

Response on success:

Status: 201

String: Message

Response on error:

Status: 404

String: Error message

Transfer money: PUT /transfer

Transfer money from 1 account to another

Request body:

JSON:

  1. {
  2. accountFrom: string,
  3. accountTo: string,
  4. sum: number
  5. }

Response on success:

Status: 200

String: "Transfer done successful"

Response on error:

Status: 400

String: Error message

Deposit: PUT /deposit

Add “sum” of money to account number “accountTo”

Request body:

JSON:

  1. {
  2. accountFrom: "",
  3. accountTo: string,
  4. sum: number
  5. }

Response on success:

Status: 200

String: Message

Response on error:

Status: 400

String: Error message

Withdraw: PUT /withdraw

Remove “sum” of money from account number “accountFom”

Request body:

JSON:

  1. {
  2. accountFrom: string,
  3. accountTo: "",
  4. sum: number
  5. }

Response on success:

Status: 200

String: Message

Response on error:

Status: 400

String: Error message
